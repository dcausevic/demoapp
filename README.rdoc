== README


* <b>System requirements</b>

  Ruby version 1.9.3+

  Rails version 4.1+

* <b>Database</b>

  Sqlite version 3.8.6+

* <b>Set up instructions</b>

  1. Unpack the project files

  2. Go into directory

     <tt>shell> cd demoapp</tt>

  3. Initialize database
  
     <tt>rake db:create</tt>
     <tt>rake db:migrate</tt>
